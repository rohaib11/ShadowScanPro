# 📦 INSTALLATION

## Windows
```powershell
git clone https://github.com/rohaib11/ShadowScanPro.git
cd ShadowScanPro
.\scripts\install-windows.ps1
```

## Linux
```bash
git clone https://github.com/rohaib11/ShadowScanPro.git
cd ShadowScanPro
chmod +x scripts/install.sh
sudo ./scripts/install.sh
```

## Docker
```bash
docker-compose up -d
```
